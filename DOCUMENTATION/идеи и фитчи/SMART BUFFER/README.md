# SMART BUFFER

Smart Buffer — это единое пространство хранения внутри системы, в которое пользователь может мгновенно отправлять любые фрагменты текста, сцены или идеи без риска их потерять.

## Функция работает максимально просто и естественно

Пользователь может:

- выделить текст и перетащить его за край листа
- или скопировать текст (Cmd+C) и сохранить в буфер

В обоих случаях фрагмент попадает в Smart Buffer.

## Поведение системы

Smart Buffer автоматически анализирует добавленный фрагмент и распределяет его по типам:

- Note — короткие мысли, реплики, заметки
- Card — более крупные фрагменты, сцены, идеи
- Alt — альтернативные версии текста, привязанные к конкретному месту

Пользователь не обязан выбирать тип вручную, но может быстро изменить его после сохранения.

## Интерфейс

Smart Buffer представлен как плавающий элемент интерфейса (Clip), который:

- появляется при копировании или взаимодействии с текстом
- позволяет быстро сохранить фрагмент одним кликом
- открывается в рабочую область сценариста

Внутри Smart Buffer элементы отображаются в виде:

- заметок
- карточек
- альтернативных версий

## Рабочая логика

- ничего не теряется при редактировании
- любой вырезанный или скопированный текст сохраняется
- все элементы можно вернуть обратно в сценарий через drag & drop
- Smart Buffer служит промежуточным слоем между написанием и структурированием

## Основная идея

Smart Buffer — это не просто буфер обмена и не заметки.

Это “память сценариста”, которая:

- сохраняет все важные фрагменты
- снижает страх редактирования
- позволяет свободно перерабатывать текст
- превращает процесс письма в работу с материалом, а не с линейным документом

## Ключевое ощущение

Пользователь не удаляет текст — он откладывает его.

И всегда может к нему вернуться.
