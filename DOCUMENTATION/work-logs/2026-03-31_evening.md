### Новые файлы
- `src/store/theme.ts` — zustand store, persist в localStorage
- `src/components/app/ThemeProvider.tsx` — data-theme + architect grid DOM
- `src/lib/themeColors.ts` — getAccentColors() / useAccentColors() для inline styles

### Три темы

#### 🎬 Cinematic (default)
Тёплый кинематографический стиль. Золотой акцент `#D4A853`, тёмные фоны, шрифт Outfit.

#### 🌆 Synthwave 80s
Ретро-футуристический неон. Розовый `#ff2d95`, циановый `#00f0ff`, фиолетовый `#b026ff`.
- Scanline overlay, neon glow, gradient bottom line на активном табе
- AI ripple: пурпурно-розово-циановый градиент

#### 📐 Architect
Монохромный wireframe. Чистый белый на чёрном `#000000`.
- Анимированная техно-сетка (2 слоя: мелкая 20px + крупная 300px, дрейф + дыхание)
- Квадратные углы (`rounded-*` → 2px)
- Моноширинный шрифт в навбаре
- Тёмная страница сценария `#222222` с noise-текстурой

### CSS Override система (~300 строк в globals.css)
Автоматическая замена цветов через `[data-theme="..."] [class*="..."]` селекторы:
- Все `#D4A853` варианты → тематический акцент
- Все `bg-white/*`, `border-white/*` → тематические тона
- emerald/sky/amber/purple → тематические варианты
- 12+ hardcoded тёмных hex → тематические фоны

## Bug Fix: Canvas → SCRIPTWRITER навигация

**Проблема**: Клик по сценарию на Canvas (/board) открывал ScriptWriterOverlay как оверлей, но URL оставался `/board`. Навбар показывал DESKTOP активным вместо SCRIPTWRITER.

**Причина**: `enterEditor()` в Canvas.tsx только ставил `editorMode.active = true`, не навигировал.

**Фикс**: Добавлен `router.push("/")` в `enterEditor()`. Страница `/` уже рендерит ScriptWriterOverlay когда есть activeProjectId. Данные скрипта в zustand store — не теряются при навигации.
