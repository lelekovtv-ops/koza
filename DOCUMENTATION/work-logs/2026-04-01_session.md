## What Was Built

### 1. Pipeline Constructor (Breakdown Studio)

**File:** `src/app/dev/breakdown-studio/page.tsx` (major rewrite)

Full visual pipeline editor built on ReactFlow. Users drag module nodes onto a canvas and wire them together to define breakdown pipelines.

**Node types:**
- **AI Module** — LLM call with system prompt, model selector, temperature slider
- **Data Node** — inputs: Scene, Style, Director, Bible
- **Output Node** — final result collector
- **Code Node** — JSON Parser, Prompt Merger (deterministic transforms)
- **ForEach Node** — per-item execution (e.g., run Prompt Composer once per shot)

**Built-in presets:**
- **Fincher** — 5 modules (Scene Analyst → Prop Scanner → Shot Planner → Prop Distributor → Prompt Composer)
- **Disney / Анимация** — 2 modules (Shot Planner → Prompt Composer)

Save/Load pipeline configurations to localStorage. Auto-save session (scene, Bible, style, images) to localStorage + IndexedDB.

---

### 2. Smart Scan for Props

Button in the Bible section of Pipeline Constructor. Scans scene text via Claude Sonnet, finds physical objects not yet listed in Bible, shows suggestion cards with accept / dismiss / accept-all actions.

---

### 3. Prop Distributor Module (Fincher pipeline)

New module inserted between Shot Planner and Prompt Composer. For each shot, decides which props are **VISIBLE** based on camera angle, shot size, and depth of field. Returns `visibleProps` with placement/size_in_frame and `excludedProps` with reasons.

---

### 4. Adaptive Prompt Composer

Context-aware prompt generation:
- `hasRefImage=true` → character name only (ref image handles appearance)
- `hasRefImage=false` → short appearance description (15 words max)
- Close-up → skip location description
- Wide shot → include location
- Props sized by importance in frame

---

### 5. LLM Reference Image Picker

Before generating an image, asks Gemini Flash which Bible refs match the prompt.

- Builds numbered catalog: `[0] ПЕРСОНАЖ: МАРИНА — appearance...`, `[1] ПРЕДМЕТ: Шкатулка — description...`
- Model returns array of indices
- Priority: Props > Main character > Secondary characters > Location
- API ref limit raised from 3 to 8 (Gemini accepts more)
- Pre-picks refs for all shots after pipeline runs (visible in UI before Generate)

---
