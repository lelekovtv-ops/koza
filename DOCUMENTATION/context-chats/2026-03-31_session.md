## Архитектурные решения

1. **Page break engine** вынесен в отдельный файл — чистая функция `calculatePageBreaks(children)`, не зависит от React
2. **Bible markers через Slate decorate** — не модифицирует DOM, работает через leaf rendering
3. **Command Bar в Zustand** — commandBarOpen state, persist настроек
4. **Ambient images в IndexedDB** — per-project ключи, rolling window, hash dedup
5. **Idle detection через window events** — keydown/mousemove/mousedown, 60s таймер
6. **Frosted glass на page backgrounds** — backdrop-filter на каждой странице A4, не на контейнере
